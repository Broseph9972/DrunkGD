# Alchaholism
(My First GD mod yayyyy)

This mod speeds up and slows down your game randomly to simulate drunkness.
it's very *customizable* and can be used to troll your friends. it has 5 presets:

- Barely noticable (to gaslight)
- Annoying (looks funny)
- Drunk (just makes levels harder)
- Wasted (changes gravity and is overall impossible to play)
- Custom (you can configure it how you want)

you can show/hide a speed viewer in the top right, and make it so when u hold down 1 key it will stay at 1x speed. you can also hide the button in the pause menu. i built this mostly to troll my friend ty. when he goes to the bathroom ill install it and put it on barely noticable to ragebait him.
